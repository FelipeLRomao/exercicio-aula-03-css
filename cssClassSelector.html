Seletor de classe (class selector)

Usamos class para poder editar vários elementos que teram a mesma estilização de uma única vez!

O seletor de classe é usado no CSS, quando queremos aplicar um estilo em um grupo de elementos HTML de página web e não em um único elemento como fizemos anteriormente. Com isso, é possível configurar parte da informação exibida em um documento HTML. A sintaxe para criar uma classe no CSS é:

.nome_da_classe { propriedade01: "valor01"; propriedade02: "valor02"; }

Deve-se colocar um ponto ( . ) antes do nome para indicar que é uma classe na folha de estilo. Esse tipo de declaração de classe no CSS pode ser aplicado em qualquer elemento na sua página HTML. Vamos ver o exemplo abaixo, que cria uma classe com o nome cor_texto com a propriedade color configurada para a cor vermelho forte (#C70000).

.cor_texto { color: #C70000; }
